# 🚀 EasyVouch Family

نظام احترافي لإدارة سيرفرات المايكروتك وتوليد الكروت والبروفايلات بطريقة ذكية وسريعة.

---

## 📌 نبذة عن المشروع

**EasyVouch Family** هي منصة متكاملة لإدارة شبكات MikroTik، مصممة لتسهيل إنشاء الكروت، إدارة البروفايلات، ومراقبة السيرفرات بكفاءة عالية.

تجمع المنصة بين السرعة، البساطة، والاستقرار لتناسب مزودي خدمات الإنترنت (ISP) والمشاريع الصغيرة والمتوسطة.

---

## ⚡ المنتجات

### ⚡ EasyVouch Mini
نظام احترافي شامل لإدارة وإنشاء كروت المايكروتك.

✔ يدعم:
- Hotspot
- User Manager
- Broadband

✔ المميزات:
- إدارة عدد غير محدود من السيرفرات  
- إنشاء كروت بسرعة عالية  
- نظام ترخيص واحد لكل جهاز  
- أداء مستقر وسريع  

---

### 🌐 EasyVouch Lite
نسخة خفيفة ومخصصة لإدارة الهوتسبوت فقط.

✔ المميزات:
- إدارة Hotspot فقط  
- أداء خفيف وسريع  
- مناسب للمشاريع الصغيرة  
- استهلاك منخفض للموارد
